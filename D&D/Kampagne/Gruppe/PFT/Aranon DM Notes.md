### Psychischer Stress
#### Stresslevel
- von 1 - 10
- erhält Stresspunkte wenn bestimmte Dinge passieren (siehe Tabelle Stresspunkte)

| Stresspunkte | Ereignis                                                 |
|:------------:| -------------------------------------------------------- |
|      +3      | erhält [[Psychischer Schaden]]                           |
|      +3      | Schaden würde den Charakter [[Bewusstlos]] werden lassen |
|      +2      | wird [[Verängstigt]]                                     |
|      +1      | fällt unter 50% [[Trefferpunktemaximum]]                 |
|      -1      | steigt über 50% [[Trefferpunktemaximum]]                 |
|      -2      | [[kurze Rast]] (-2)                                      |
|      -5      | [[lange Rast]] (-5)                                      |

#### Stresstest
- fällig wenn das Stresslevel auf einmal um mehr als 2 auf einmal Punkte erhöht wird
- [[Rettungswurf]] auf [[Weisheit]] gegen [[Schwierigkeitsgrad|SG]] 5 + Stresslevel
- wenn Misserfolg erfolgt unbewusste Teleportation anhand Tabelle Stresslevel

| W10 + Stresslevel | Ereignis                                                                                |
|:-----------------:| --------------------------------------------------------------------------------------- |
|      1 - 10       | wird an eine beliebige Stelle im gleichen Bereich teleportiert (nach Wahl des Spielers) |
|      11 - 12      | wird an eine beliebige Stelle im gleichen Bereich teleportiert (nach Wahl des DM)       |
|      13 - 14      | tbd                                                                                     |
|      15 - 16      | tbd                                                                                     |
|      17 - 18      | tbd                                                                                     |
|      19 - 20      | Wird in die nächste größere Stadt teleportiert                                          |

# `=this.file.name`
Fügt ein Angriff oder Effekt einer Kreatur oder einem Gegenstand Schaden zu wird ein [[Schadenswurf]] ausgeführt.
Welche und wie viele Würfel geworfen werden wird vom jeweiligen Effekt bestimmt.

## Waffenangriffe
Bei Waffenangriffen wird das [[Attribute]] das auf den [[Angriffswurf]] angewendet wurde (meist [[Stärke]] oder [[Geschicklichkeit]]) auch auf den Schaden angerechnet.

## Mehrere Ziele
Fügt ein Zauber oder Effekt mehreren Zielen Schaden zu, wird der Schaden einmal für alle Ziele ausgewürfelt.

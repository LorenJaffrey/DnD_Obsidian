# `=this.file.name`
Eine [[Runde]] stellt eine Zeitspanne von ca. 6 Sekunden in der Spielwelt dar. 
Innerhalb einer [[Runde]] hat jeder Kampfteilnehmer einen [[Zug]].
Sobald jeder seinen [[Zug]] beendet hat, geht der Kampf mit der nächsten [[Runde]] weiter.

**1× pro Runde-Fähigkeiten.**  
Diese werden zurückgesetzt, sobald dein eigener [[Zug]] wieder beginnt.
